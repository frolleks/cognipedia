# article-creator

This is the source code for the article creator LLM agent. It uses Tavily for web search, and Vercel's AI SDK for building the agent.

## Running locally

### Prerequisites

- OpenRouter API key
- Tavily API key

### Steps

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts "any topic you want it to make an article on"
```
